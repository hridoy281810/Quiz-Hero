.forEach(), .map(), .filter() , find() .... What's the difference?

Answer:

  • Map(): If you want to get propertynames from an array through map then the property names will be output as an array.
  • forEach(): If I don't want to return an array, I use forEach
  • filter(): The filter works according to the condition. The data that matches the condition will be output.
  • find(): Find and Filter work in very similar terms. But find will only output the first data that matches.
