what is difference between local storage session storage and cookies?

Answer:

  • Local Storage: The data that is permanently stored in the local store remains after the refresh, and even if the browser is closed and the browser is opened again, the data remains, so it can be said that the data is permanently stored in the local storage.
  • Session Storage:Data is stored temporarily in session storage. Closing the browser and opening it again clears the data
  • Cookie: Cookie data is stored on server side or client side. During a web browsing session, data is stored as data stings.
